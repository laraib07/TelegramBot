# Requirements

* Python 3.*
* Python-telegram-bot 12.8

# Commands

Command        | Action
:-------------:|:-----------------:
/help          |  get inline help
/timer         |  set timer
/unsettimer    |  unset timer
/reminder      |  set reminder
/unsetreminder |  unset reminder
